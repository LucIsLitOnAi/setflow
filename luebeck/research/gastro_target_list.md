## RESEARCH NOTES

**HowToVerify contacts:**
1. Visit website → find "Kontakt" page → get email + phone + manager name
2. Google My Business listing → check for "Inhaber" mention or response signatures on reviews
3. LinkedIn search: "[business name] Lübeck" → often find owners for mid-size hotels
4. OpenTable / TheFork / Yelp — cross-reference reservation platforms they're already on

**Digital Presence Scoring Guide:**
- **5:** Own booking system + active social + many Google reviews + fast GMB responses
- **4:** Good website + GMB + some social activity
- **3:** Decent website but outdated, moderate reviews
- **2:** Basic website, few reviews, no social
- **1:** No website or placeholder only

Low score = **higher pain** → easier sell on digitization
